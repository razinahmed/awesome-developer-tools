# Contributing to Awesome Developer Tools

We love contributions from the community! Here's how you can add your favorite tool to the list.

## Guidelines

- **Search first**: Ensure the tool isn't already on the list.
- **Quality over quantity**: Only submit tools that you personally use and find exceptional.
- **Description**: Provide a clear, short description of what the tool does (avoid marketing speak).
- **Format**:
  ```markdown
  * [Tool Name](https://link-to-tool.com) - A short description of the tool ending with a period.
  ```

## Making a Pull Request

1. Fork the repository.
2. Create a new branch (`git checkout -b add-amazing-tool`).
3. Add your entry to the appropriate section in `README.md`.
4. Commit your changes (`git commit -am 'Add Amazing Tool'`).
5. Push to the branch (`git push origin add-amazing-tool`).
6. Open a Pull Request.

Thank you for helping make this list awesome! 🚀
